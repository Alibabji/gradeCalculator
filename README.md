# Functions
## Grade by Percent
- Calculates average grade when marks are inputted in percent
- Range: 0~100%
## Grade by letter
- Calculates average grade when marks are inputted in letter
- Type: D-, D, D+, C-, C, C+, B-, B, B+, A-, A, A+
## Grade by points
- Calculates average grade when user inputs x out of y
	- eg. 18/20 --> 90%
	- eg. 17/17 --> 100%
	- eg. 21/20 --> 105%
